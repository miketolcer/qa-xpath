XPath & UI Testing

Introduction

In this lab, you’ll be practicing selecting UI elements using XPath and running UI tests.

Instructions

Summary

Let’s write a UI test for GitHub.com! You’ll want your test to visit GitHub and navigate around.

npm run test

Throughout the lab, you will be using npm run test to run your tests.

Steps

  1. Download the lab materials from Frodo

  2. Run npm i in that directory

  3. Open the google.test.js file for a quick refresher (find the selector for line 10)

  4. Open the github.test.js file from that folder

  5. Open up GitHub so that you can use the inspector

  6. Find CSS or XPath selectors for the elements you’ll need to complete this flow: open the user dropdown menu from the top right, visit your profile, click to edit your profile, add something to your bio, and save the change

  7. In your test file, write a test that completes that flow

  8. Come up with another flow and write a test for that flow (Challenge: only use XPath selectors)
